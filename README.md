# Coins V1.1
 A currency quote app where you can check the price of the main currencies on the market.

### Market Coins

- Dolar (US, canadian, Australian and New Zealander)
- EURO
- Peso (Argentine, Mexican and Chilean)
- Pound (UK)
- Swiss Franc
- Russian Ruble
- Renminbi (China)
- Iene (Japan)
- Dirham (United Arab Emirates)
- South African Rand
