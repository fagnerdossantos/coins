# Coins V3
 A currency quote app where you can check the price of the main currencies on the market.

### Market Coins

- Dolar (US and Australian)
- EURO
- Pound (UK)
- Renminbi (China)
- Iene (Japan)
- Argentine peso

> And two crypto
